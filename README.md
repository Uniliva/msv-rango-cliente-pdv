# msv-rango-cliente-pdv

## Descrição
 Micro serviço responsável pela recuperar pedidos e salvar na base de dados
 
## Tecnologias 
- SpringBoot
- Java
- Kafka
- Docker
- Lombok
- Flaway
- PostgreSql
- JDBC Template
 
## Configuração Docker
- IP_KAFKA_SERVER variavel do ambiente do kafka server
- POSTGRE_URL
- POSTGRE_PORT
- POSTGRE_DATABASE
- POSTGRE_USER
- POSTGRE_PASS
